# LegendaryMissions

**LegendaryMissions** is the standard set of missions and reusable addons built
on `sbs_utils` (fleets, comms trees, docking, damage, upgrades, consoles,
gamemaster, and more). It depends on `sbs_utils`; `sbs_utils` never depends on it.

This section covers **playing**, **hosting**, and the **addon reference**. Its
pages live in the [LegendaryMissions repo](https://github.com/artemis-sbs/LegendaryMissions)
and are stitched into this site at build time (see the docs build notes).

Mission **authoring** &mdash; the MAST language, GUI/comms/science/AI recipes, and
the API &mdash; lives in the rest of this site, since it applies to any mission.
